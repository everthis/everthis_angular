git使用教程(注：OS X与Windows下使用方法基本一致。)
  1. 到http://git-scm.com/下载并安装git客户端
  2. ssh-keygen -t rsa -C "Your email address"
  3. 将生成的.pub文件内容填到如下key位置
  4. git config --global user.name "username"
  5. git config --global user.email "Your email address"
  6. mkdir test
  7. cd test
  8. git init
  9. touch README.md
  10. git add README.md
  11. git commit -m "first commit"
  12. git remote add origin Your_git_repo
  13. git push -u origin master
  14. 获取最新版本：git pull
